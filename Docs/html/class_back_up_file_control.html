Descripción detallada

La función principal de este componente es la restauración de los datos de las posiciones de la simulación en caso de que un comando en el Scorbot real bloque el programa. Las posiciones fuera del alcance o que no se pueden cumplir las restricciones de pitch, se pondrán en una posición por defecto. ANTES de CARGAR los datos SELECCIONAR el SCORBOT deseado. (normal|relative) name x y z p r (relativeTo).
